欢迎光临管理平台

通过管理平台操作，你可以对发布的信息进行批量管理。


快捷管理菜单

  • 用户/空间

    编辑用户的积分、用户组、管理空间信息、删除用户

  • 事件

    对"我在做什么"进行批量删除

  • 日志

    对日志进行批量删除、编辑

  • 相册

    对发布的相册进行批量删除

  • 图片

    对发布的图片进行批量删除

  • 评论

    对发布的评论进行批量删除

  • 话题

    对选吧上面发布的话题进行批量删除、精华、置顶

  • 回帖

    对选吧上面发布的回帖进行批量删除

  • 标签

    对站点的标签进行删除、关闭、合并

  • 选吧

    对站点的选吧进行删除、关闭、合并

  • 模版样式

    对站点共享的模版样式进行批量审核、删除


程序版本

  • 您当前使用的程序版本：UCenter Home $statistics[version] ( $statistics[release] )

系统统计信息

  • 操作系统: $os
  • 数据库版本: $statistics[mysql]
  • 上传许可: $fileupload
  • 数据库尺寸: $dbsize
  • 附件尺寸: $attachsize
  • 空间数: $statistics[spacenum]
  • 日志数: $statistics[blognum]
  • 相册数: $statistics[albumnum]
  • 话题数: $statistics[threadnum]
  • 事件数: $statistics[feednum]
